# Security

Raid IV Odds is a static, client-side calculator. It has no accounts, backend database, payments, or app-managed secrets.

## Reporting

Please report security or privacy issues privately to the repository owner. Include:

- What happened.
- Steps to reproduce it.
- Browser and device details.
- Whether it affects the hosted app, repository, or both.

## Notes

- Calculator inputs are processed in the browser.
- Public pull requests do not change the live site unless they are reviewed, merged, and deployed by the repository owner.
- Direct write access should stay limited to trusted collaborators.
